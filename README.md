## Run Locally

Clone the project

```bash
  git clone https://github.com/thurdelima/node-jest-api-books
```

Go to the project directory

```bash
  cd node-jest-api-books
```

Install dependencies

```bash
  yarn install or yarn
```

Configure .ENV vars


Run migration

```bash
  npx sequelize-cli db:migrate

```

And

```bash
  yarn start

```
